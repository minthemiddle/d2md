# d2md

Reads markdown diary from Mac clipboard.  
Saves to individual markdown files in output folder.
Companion app for [d diary app](https://github.com/minthemiddle/d).

## Format

Your markdown needs to have a certain format.

````markdown
# YYYY-MM-DD

Body
Of 
Diary
Entry

# YYYY-MM-DD

Body 2
...
```
